# CS3-DS4002

This repository contains all materials needed to complete the case study on sentiment shifts in vaccine-related tweets by theme. It includes the dataset, code, rubric, hook document, and supporting reference materials.

---

## Important Information

The included PDF files contain:
- The **Hook Document**, which introduces the scenario and frames the student task  
- The **Rubric**, which outlines grading expectations for the code, analysis, and written report  

Students should review both documents before beginning the assignment.

---


## Materials

The **Materials** folder contains the data, the code, and the articles for reference. 

These include:
- Articles on vaccine misinformation and public health communication  
- Background resources on sentiment analysis and social media data  
- Jupyter Notebook
- CSV File
